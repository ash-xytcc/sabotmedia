# Content Pipeline Wave Notes

This pragmatic pass covers:

## Phase 27
A source bridge for native content entries:
- source type
- source label
- source URL
- external ID
- notes

This is not full sync. It is provenance tracking and future connector readiness.

## Phase 28
A transcript bridge:
- transcription status
- audio source URL
- full transcript storage
- transcript notes

This is not a full job runner. It is the storage and workflow shape needed before automated transcription lands.

## Phase 29
Public search:
- native updates
- imported archive pieces
- transcript/excerpt searchable text where available

This is app-level pragmatic search, not semantic/vector search.
